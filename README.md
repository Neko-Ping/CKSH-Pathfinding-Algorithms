# 成功高中室內立體導航實測

## 專案概述：
本專案為成功高中數理資優班的資訊科獨立研究之一。
以本校綜合大樓 1F 至 8F 為界，透過圖論（Graph Theory）建構三維座標模型，方便使用者在複雜的大樓內找尋地點，同時也將平面圖數位化並加入距離關係。
此外，也透過加入三種演算法的選擇與重要資訊如執行時間的輸出，去方便比較影響最短路徑演算法的因素等。

## 核心技術
- **演算法實作**：加入 Dijkstra、Bidirectional Dijkstra 與 A* 三種最短路徑演算法，以分情況選擇最適配之演算法。
- **3D 座標建模**：定義節點之 (x, y, z) 座標，並納入電梯與樓梯之加權計算，避免程式被無條件導向電梯，違反現實直覺。
- **中繼站選擇與自動排序**：依此功能能夠增加路徑可變性，並在實驗時幫助本組固定路徑，以針對單一變因進行實驗。

## 實驗結論
實測顯示， A* 演算法在室內導航中具備最佳執行速度，而 Bidirectional Dijkstra 則在路徑精確性上表現穩定。而節點數對執行時間呈二次方關係，卻與路徑長並無直接關聯。
而實體網頁之網址則為https://neko-ping.github.io/CKSH-Pathfinding-Algorithms/
